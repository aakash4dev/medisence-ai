## Getting Started

### First Time Setup
1. **Open the Application**
   - Navigate to `http://localhost:5000`
   - The homepage will load with the chat interface

2. **Optional: Create an Account**
   - Click "Sign In" in the top right
   - Enter your phone number
   - Verify with OTP
   - Complete your profile

### Navigation
- **Home**: Main chat interface
- **Dashboard**: View appointments and health records
- **Profile**: Manage your information
- **Settings**: Customize preferences

## Using the Chatbot

### Starting a Conversation
1. Click the **blue chat icon** in the bottom-right corner
2. Type your symptoms or medical question
3. Press **Enter** or click **Send**
4. Wait for AI analysis (1-2 seconds)

## Tips & Best Practices

### For Accurate Diagnosis
1. **Be Detailed**: More information = better analysis
2. **Use Plain Language**: No medical jargon needed
3. **Mention Timeline**: When did symptoms start?
4. **Note Changes**: Any improvement or worsening?
5. **List Medications**: Include current medications

### Privacy & Security
1. **Don't Share**: Never share login credentials
2. **Secure Device**: Use password/biometric lock
3. **Log Out**: Log out on shared devices
4. **Review Permissions**: Check app permissions regularly

### Regular Health Monitoring
1. **Weekly Check-ins**: Record vitals weekly
2. **Symptom Diary**: Track recurring issues
3. **Medication Log**: Note when taken
4. **Follow-ups**: Track doctor recommendations

### When NOT to Use the App
‚ùå **Don't use for:**
- Active emergencies (Call 911 first)
- Prescription requests
- Formal medical diagnosis
- Legal medical documentation
- Insurance claims

‚úÖ **DO use for:**
- Preliminary symptom assessment
- Health tracking
- Appointment management
- General health guidance
- Medical information

## Troubleshooting

### Chat Not Responding
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Try different browser

### Can't Book Appointment
1. Verify all fields filled
2. Check date is in future
3. Ensure phone verified
4. Contact support

### Images Not Uploading
1. Check file size (< 10MB)
2. Use supported formats (JPG, PNG)
3. Check internet speed
4. Try smaller image
